body - Wooden Ladder(Деревяні сходи)

Unit test: Перед початком встановлення перевірити окремі деталі(частини)
сходів на наявність дефектів, якщо ж такі дефекти не суттєві наприклад сучки то можна їх 
вважати як фітча.

Integration test - при складанні різних деталей сходів в одне ціле протестувати
на точність усі виступи і врізи для кращого кріплення,  атакож правельний монтаж і
монтажні засоби кріплення.

System tes - потрібно перевірити чи достатньо місця виділено під сходи. 
Перевірити чи співпадають розміри сході із розмірами вхідного і вихідного отворів, 
перевірити чи естетично у приміщення такі сходи.

Smoke test - При неретельному огляді схдів превірити чи існують видимі дефекти,
які можуть завадити експлуатуванні сходів, якщо ж такі дефекти існують звернутись
до виробника згідно прав споживача з вимогою замінити або ж виправити дефекти які не 
задекларовані в документації як фітча.

Functional test - Пойтись по сходах вгору, вниз, превірити чи зручно ходити, чи нічого
не мішає пресуванню. Перевірити чи нормально пересуватись по сходах людям з високим зростом та 
великою вагою.

GUI test - перевірити чи відповідають сходи стандартам ДСТУ та чи вписуються в інтерєр.

Globalization test - Перевірити чи зручні у використанні сходи для людей з обмеженими можливостями 
дітям, та людям похилого віку.

Localization test - Перевірити чи інструкція для встановлення та використання написана
мовою рідною мовою клієнта або ж мовою яку він розуміє.

Usability test - Перевірити зручність використання сходів одночасно декількома людьми, наприклад:
одна людина піднімається по сходах а інша спускається. Якщо ж будуть проблеми з пресуванням 
скільки часу займе вирішення колізії. Також потрібно не забувати про Emotional response
чи порадить одна людинина використовувати такі сходи іншому.

Regression testing - Якщо був знайдений у сходах баг, чи після деякого часу використання
виник дефект потрібно викликати майстра, після чого протестувати повторно.

Performance, Load Testing - Перевірити скільки часу знадобиться для проходженя сходів під 
різним їх завантаженням. Наприклад: 2 людини стоять на сходах розмовляють, 1 - піднімається
і одночасно 1 спускається, скільки часу знадобиться для проходження сходів
якщо цей час вважається нормальним то тест пройдено.

Acceptance testing - провести випробування сходів за уклієнтом, сторонніми людьми.
