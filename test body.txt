body - Wooden Ladder(Деревяні сходи)

Smoke test - При неретельному огляді схдів превірити чи існують видимі дефекти,
які можуть завадити експлуатуванні сходів, чи візуальний розмір і колір сходів
ідентичний як у документації.

Unit test: Перед початком встановлення перевірити окремі деталі(частини)
сходів на наявність дефектів, якщо ж такі дефекти не суттєві наприклад сучки то можна їх 
вважати як фітча, перевірити чи присутні монтажні елементи(гайки, шурупи, інше), 
чи потрібних розмірів. 

Integration test - при складанні різних деталей сходів в одне ціле протестувати
на точність усі виступи і врізи для кріплення кріплення, чи різні елементи сходів 
підходять одне до іншого.

System test - потрібно перевірити чи достатньо місця виділено під сходи. 
Перевірити чи співпадають розміри сході із розмірами вхідного і вихідного отворів
чи стоятть сходи твердо на поверхні.

Functional test - Пойтись по сходах вгору, вниз, превірити чи можна по них ходити.

Usability test - Перевірити чи нічого не мішає пересуванню по сходах, перевірити чи висота 
та ширина сходинок відповідає станлартам та чи вона однакова, якщо сходи мають поруча 
перевірити чи воно знаходитья на належній висоті від сходів, та чи зручно за нього триматись.
Також перевірити чи не мають сходи спецефічного запаху(клею, лаку, фарби).

Performance - Одночасно пройтись по сходах декільком люлям, перевірити чи зручно буде швидко
підніматись або опускатись по сходах.

Load Testing - Перевірити скільки часу знадобиться для проходженя сходів під 
різним їх завантаженням, якщо цей час вважається нормальним то тест пройдено.

GUI test - перевірити чи відповідають сходи стандартам ДСТУ та чи вписуються в інтерєр, 
чи колір та вигляд відповідає документації.

Globalization test - Перевірити чи зручні у використанні сходи для людей з обмеженими можливостями 
дітям, та людям похилого віку.

Localization test - Перевірити чи інструкція для встановлення та використання написана
мовою рідною мовою клієнта або ж мовою яку він розуміє.
